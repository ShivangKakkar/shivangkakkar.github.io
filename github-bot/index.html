How to use GitHub Bot?

  1. Visit your GitHub repository using your web browser.

  2. Go to the Settings Tab

  3. Tap on Webhooks

  4. Click on Add Webhook

  5. Paste your webhook url. You can get it from GitHub Bot by sending command /url

  6. GitHub will ask you Which events would you like to trigger this webhook.
    It's recommended to use Send me Everything to get all updates. If you don't want all updates, choose other options accordingly.

  7. Leave everything else with default values. Click on Add Webhook

  8. You have successfully linked the bot. Updates will be sent to your private chat unless group is set. To set the group use /group <chat id or username>.

    A message will also be sent in your chat telling if the process was successful.
